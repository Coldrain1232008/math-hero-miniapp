# 小程序图标

已为你生成了一个 SVG 格式的小程序图标。

## 图标位置
`tools/app-icon.svg`

## 图标设计
- **主题**：几何三角形 + 铅笔 + 书本
- **配色**：紫色渐变背景（#6C63FF → #4F46E5） + 金黄色元素
- **装饰**：数学符号（∑、π、∫）
- **文字**：底部"数学英雄"

## 转换为 PNG
小程序注册需要 PNG 格式，请按以下步骤转换：

### 方法一：在线转换（推荐）
1. 访问：https://svgtopng.com/
2. 上传 `tools/app-icon.svg`
3. 设置输出尺寸：512 × 512
4. 下载转换后的 PNG 文件
5. 在小程序注册时上传这个 PNG

### 方法二：使用 Canva/即时设计
1. 打开 Canva (可画) 或 即时设计
2. 新建 512×512 画布
3. 导入 `app-icon.svg` 或参考设计重新绘制
4. 导出为 PNG

### 方法三：专业工具
如果你有 Adobe Illustrator 或 Sketch：
1. 打开 SVG 文件
2. 导出为 PNG，尺寸 512×512

## 其他尺寸建议
小程序注册需要以下尺寸的图标：
- **小程序图标**：512×512
- **小程序码**：会自动生成

如果需要其他尺寸（如网站展示用），可以导出多个版本。
