# 学生名单导入模板

## Excel 文件格式说明

批量导入学生到班级时，Excel 文件需要包含以下列：

| 列名 | 说明 | 必填 | 示例 |
|------|------|------|------|
| 姓名 | 学生姓名 | 是 | 张三 |
| 学号 | 学生学号或唯一标识 | 是 | 2023001 |

## 示例格式

| 姓名 | 学号 |
|------|------|
| 张三 | 2023001 |
| 李四 | 2023002 |
| 王五 | 2023003 |

## 使用步骤

1. 准备符合上述格式的 Excel 文件（.xlsx 或 .xls）
2. 打开小程序，教师端进入「班级管理」
3. 点击「批量导入学生」
4. 选择准备好的 Excel 文件
5. 系统会自动解析并添加到班级

## 注意事项

- 学号必须是唯一的，不能重复
- 姓名将用于匹配成绩导入
- 导入的学生初始状态：
  - 累计经验：0
  - 等级：1
  - 需要通过小程序登录创建角色

## Excel 文件示例

```
姓名	学号
张三	2023001
李四	2023002
王五	2023003
赵六	2023004
钱七	2023005
```
