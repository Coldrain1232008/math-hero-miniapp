# 成绩导入模板

## Excel 文件格式说明

上传的 Excel 文件需要包含以下列（顺序不限）：

| 列名 | 说明 | 示例 |
|------|------|------|
| 姓名 | 学生姓名，必须与系统中注册时使用的姓名一致 | 张三 |
| 排名 | 本次考试成绩排名（数字，第1名、第2名...） | 5 |
| 考试名称 | 本次考试的名称（可选） | 期中考试 |

## 示例格式

| 姓名 | 排名 | 考试名称 |
|------|------|----------|
| 张三 | 5 | 期中考试 |
| 李四 | 12 | 期中考试 |
| 王五 | 1 | 期中考试 |

## 经验值计算规则

- **排名 1**：150 EXP
- **排名 2**：145 EXP
- **排名 3**：140 EXP
- ...依此类推
- **排名 50+**：50 EXP（最低）

计算公式：`经验值 = 155 - 排名`（最低50，最高150）

## 注意事项

1. 文件必须是 `.xlsx` 或 `.xls` 格式
2. 第一行必须是表头（包含"姓名"和"排名"列）
3. "姓名"必须准确匹配系统中学生创建角色时填写的姓名
4. 排名必须是正整数
5. 同一学生可以多次导入，每次都会累加经验值

## Excel 文件示例

你可以直接复制下表到 Excel 中保存使用：

```
姓名	排名	考试名称
张三	5	期中考试
李四	12	期中考试
王五	1	期中考试
赵六	8	期中考试
钱七	3	期中考试
```
