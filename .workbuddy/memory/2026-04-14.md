# 2026-04-14 工作日志

## drawGacha where().update() 返回结构 Bug 修复

**问题**：抽卡时显示"今日抽卡次数已用完"，但页面次数正常-1。
**根因**：`where().update()` 返回的是 `{ stats: { updated: N } }`，而不是 `{ updated: N }`。代码判断 `!updateRes.updated` 永远为 true（因为 `updateRes.updated` 是 undefined），导致所有抽卡请求都被当成失败处理。
**修复**：改为 `updateRes.stats.updated` 判断。
**教训**：微信云数据库 `doc().update()` 和 `where().update()` 返回结构不同！
- `doc().update()` → `{ updated: 1 }`
- `where().update()` → `{ stats: { updated: 1 }, errMsg: "collection.update:ok" }`
