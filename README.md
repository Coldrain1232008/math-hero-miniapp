# 数学英雄 - 微信小程序

> 将学生成绩转化为角色成长，通过天赋、属性和等级系统激励初中生持续进步。

---

## 项目结构

```
math-hero-miniapp/
├── miniprogram/                 # 小程序前端
│   ├── pages/
│   │   ├── login/               # 登录页（学生/教师）
│   │   ├── create-character/    # 角色创建（取名+天赋随机）
│   │   ├── character/           # 角色主页（雷达图+属性+日志）
│   │   ├── ranking/             # 班级排行榜

│   │   ├── teacher-score/       # 课堂打分
│   │   ├── teacher-upload/      # 成绩导入
│   │   └── teacher-class/       # 班级管理
│   ├── components/
│   │   └── radar-chart/         # 六维雷达图组件
│   └── utils/
│       └── gameData.js          # 游戏核心数据（天赋、升级表、计算函数）
└── cloudfunctions/              # 云函数
```

## 天赋系统

共 **6 大类 × 24-30 种子类型**，天赋影响每级属性成长率：

| 大类 | 特点 | 代表子类 |
|------|------|---------|
| 探索者 | 智识+灵感高成长 | 星图探索者、海图探索者 |
| 铸造者 | 专注+毅力高成长 | 寒冰铸造者、光明铸造者 |
| 编织者 | 表达+灵感高成长 | 语言编织者、梦境编织者 |
| 守护者 | 心志高成长 | 圣盾守护者、黎明守护者 |
| 引导者 | 智识+表达均衡 | 光之引导者、时之引导者 |
| 突破者 | 某一属性极高 | 黄金突破者、钢铁突破者 |

## 升级系统

- 最大等级：**20级**
- 升级所需经验：前期低（Lv1→2 仅需50EXP），后期高（Lv19→20 需5280EXP）
- 一个学期（18周）合理可达 **10-15级**

### 经验来源

| 来源 | 经验值 |
|------|--------|
| 老师课堂打分 | +1 EXP（每次） |
| 考试排名第1 | +150 EXP |
| 考试末名 | +50 EXP |
| 中间名次 | 线性插值 150→50 |

## 部署步骤

### 1. 注册小程序 & 开通云开发

1. 登录 [微信公众平台](https://mp.weixin.qq.com) 注册小程序
2. 在微信开发者工具中开通**云开发**
3. 记录云环境 ID，填入 `app.js` 中的 `env: 'your-env-id'`

### 2. 创建云数据库集合

在云开发控制台创建以下集合：

| 集合名 | 说明 |
|--------|------|
| `classes` | 班级信息（secretKey、teacherKey、name） |
| `students` | 学生角色数据 |
| `expLogs` | 经验值记录日志 |

**创建第一个班级**（手动在控制台插入一条）：
```json
{
  "name": "初一3班",
  "secretKey": "MATH2024",
  "teacherKey": "TEACHER2024"
}
```
> 学生用 `MATH2024` 登录，教师用 `TEACHER2024` 登录

### 3. 部署云函数

在微信开发者工具中，右键每个云函数目录 → **上传并部署**：
- `addExp`
- `createStudent`
- `parseExcel`（需先 `npm install`）
- `importStudents`

### 4. 配置数据库权限

在云开发控制台，将各集合的权限设置为：
- `students`：所有用户可读，仅创建者可写（安全规则用云函数覆盖）
- `expLogs`：所有用户可读，仅创建者可写
- `classes`：所有用户可读，仅管理员可写

### 5. 上传小程序代码

在开发者工具中上传代码 → 提交审核（如需发布）

## 使用流程

### 教师

1. 用**教师密钥**登录
2. 进入**班级管理** → 导入学生名单（批量输入姓名）
3. 系统为每位学生生成**6位学生密钥**，分发给学生
4. 上课时进入**课堂打分** → 点击学生名字 → 一键+1经验
5. 考试后进入**导入成绩** → 上传Excel → 确认发放

### 学生

1. 用**学生密钥**登录
2. 首次登录 → 创建角色（取名、选头像、随机天赋）
3. 查看**我的英雄**：雷达图、属性值、等级、经验记录
4. 查看**英雄榜**：班级排名

---

## Excel 成绩格式

```
学生姓名  |  分数
张三      |  92
李四      |  85
王五      |  78
```
- 第一行为表头
- 姓名需与系统中学生姓名完全一致
- 系统自动按分数排名，计算各人经验值
